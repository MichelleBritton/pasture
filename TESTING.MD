|User Story:             |As a user, I want the navigation to be clear and visible so that I can easily navigate through the website|
|:--                     |:--                                                                                                       |
|**Acceptance Criteria:**|The site navigation should be easily accessible and consistent on every page so the user can visit all pages|
|**Summary**             |- When a user visits the site, the navigation is at the top of every page<br>- When a user scrolls down the page, the navigation stays fixed at the top of the screen<br>- When a user navigates through the pages on the site, the navigation remains in the same place<br>- When a user is logged in, the Register and Login links are replaced with the appropriate profile (either Profile or Staff Management) link and a log out link|     
|**Outcome**             |All features tested and passed.|
